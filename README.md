# Customer Churn Prediction using Logistic Regression (From Scratch)

## Overview
This project implements Logistic Regression from scratch in Python without using Scikit-learn. The model predicts whether a customer is likely to churn based on Monthly Charges and Tenure.

## Features
- Logistic Regression implemented from scratch
- Custom Sigmoid function
- Binary Cross-Entropy Loss
- Gradient Descent optimization
- Customer churn prediction

## Technologies Used
- Python
- Pandas
- Math Library

## Dataset
The dataset contains the following columns:
- MonthlyCharges
- Tenure
- Churn

## How to Run

1. Clone the repository
2. Install pandas
3. Run:

```bash
python main.py
```
